# Vault Index

This vault is private mutable user state for the current environment/profile.

## Start here

- [[conventions]] — folder semantics, note status, and linking conventions.
- [[maintenance]] — health checks and curation workflow.

## Main areas

- `daily/` — dated scratchpad and capture ledger.
- `inbox/` — unintegrated candidate notes.
- `working/` — active/provisional task context.
- `memory/` — curated durable knowledge.
- `outputs/` — generated plans, reports, and summaries.
- `templates/` — Obsidian/Neovim templates.
- `attachments/` — binary assets and pasted media.
